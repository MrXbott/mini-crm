
## Мини-CRM для распределения обращений клиентов между операторами по источникам

### Как запустить
В проекте использован Poetry (подразумевается, что он у вас уже установлен на компе). 
Перед запуском проекта нужно установить зависимости следующей командой:

   `poetry install`

Можно запустить проект следующей командой (без предварительной активации вирт. окружения):

  `poetry run uvicorn mini_crm.main:app --reload`
